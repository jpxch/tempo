# Tempo Design System

## Personality

- Calm
- Intelligent
- Premium
- Creative
- Trustworthy

## Avoid

- Corporate
- Sterile
- Cluttered
- Gimmicky
- Startup-bro

## Design Philosophy

Tempo should feel like a personal assistant, not a spreadsheet.

Information should feel organized and approachable.

The user should feel less overwhelmed after opening Tempo.

## Visual Direction

- Dark mode first
- Soft contrast
- Spacious layouts
- Strong typography
- Purposeful color accents
- Motion should be subtle and meaningful

## Core Principle

Clarity over cleverness.

## Project Color System

Projects should support assigned colors.

A project color should appear consistently across:

- project cards
- reminders
- notes
- events
- payments
- calendar items

The goal is instant recognition without needing to read every label.
